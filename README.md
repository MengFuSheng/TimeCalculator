# 时间计算器 - 功能与技术说明

## 📱 应用概述

**时间计算器** 是一款专为Android平台设计的实用工具应用，旨在帮助用户轻松处理各种时间计算需求。无论是计算项目时长、规划未来日程，还是进行时间单位换算，都能在本应用中快速完成。

## ⭐ 核心功能模块

### 1. **时间间隔计算**
- **功能**：精确计算任意两个时间点之间的时长
- **特点**：
  - 支持日期和时间选择器，避免手动输入错误
  - 可切换12小时制/24小时制显示
  - 结果展示精确到秒，支持天、小时、分钟、秒多种单位
  - 自动识别时间先后顺序，智能提示

### 2. **时间点推算**
- **功能**：基于基准时间推算未来或过去的时间点
- **特点**：
  - 灵活的时间增减操作（支持加/减）
  - 多时间单位选择（秒、分钟、小时、天、周、月、年）
  - 实时计算结果预览
  - 常用时长预设，一键快速计算

### 3. **时间单位换算**
- **功能**：在不同时间单位间进行快速转换
- **特点**：
  - 支持10种时间单位：纳秒、微秒、毫秒、秒、分钟、小时、天、周、月、年
  - 实时双向换算，支持单位交换
  - 三种显示精度（整数、2位小数、6位小数）
  - 换算历史记录，最多保存20条记录

## 🛠️ 技术架构

### **开发语言与环境**
- **开发语言**：Java
- **最小支持版本**：Android 7.0 (API 24)
- **开发工具**：Android Studio
- **架构模式**：模块化Activity设计

### **核心技术选型**

| 技术组件 | 选型理由 | 版本 |
|---------|---------|------|
| **时间处理库** | ThreeTenABP (JSR-310 backport) | 1.4.9 |
| **UI框架** | Material Design组件 | 1.13.0 |
| **布局系统** | ConstraintLayout | 2.1.4|
| **布局系统** | CardView | 1.0.0 |
| **数据持久化** | SharedPreferences + JSON | 系统原生 |

### **兼容性设计**
- **低版本兼容**：通过ThreeTenABP库在现代时间API和旧版Android系统间架起桥梁
- **多屏幕适配**：使用ConstraintLayout确保不同屏幕尺寸的良好显示
- **内存优化**：合理管理Activity生命周期，避免内存泄漏

### **关键技术实现**
1. **时间选择器集成**：原生DatePickerDialog和TimePickerDialog，支持12/24小时制
2. **实时计算引擎**：基于ThreeTenABP的Duration和Period类进行精确时间计算
3. **数据持久化**：SharedPreferences保存用户偏好，JSON格式存储历史记录
4. **错误处理**：全面的异常捕获和用户友好提示
